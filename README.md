# 📊 Poll Results Visualizer (Google Colab Edition)

## 🚀 Overview

The **Poll Results Visualizer** is a data analytics project that transforms raw poll/survey data into meaningful insights using Python.

This version is built entirely on **Google Colab**, making it easy to run without any local setup.

---

## 🎯 Objective

To build a system that:

* Generates or imports poll/survey data
* Cleans and processes responses
* Performs statistical analysis
* Visualizes results using charts
* Extracts actionable insights

---

## 🧠 Key Features

* 📥 Synthetic poll data generation (realistic simulation)
* 🧹 Data cleaning and preprocessing
* 📊 Vote count & percentage analysis
* 📈 Visualizations:

  * Bar Chart
  * Pie Chart
  * Trend Analysis
  * Region-wise Comparison
* ☁️ Fully executable on Google Colab
* 🧾 Insight generation

---

## 🛠️ Tech Stack

## ▶️ How to Run (Google Colab)

1. Open Google Colab
   https://colab.research.google.com

2. Upload the notebook
   `Poll_Results_Visualizer.ipynb`

3. Run all cells:

   ```
   Runtime → Run All
   ```

4. Outputs:

   * Charts displayed in notebook
   * Insights printed below cells
